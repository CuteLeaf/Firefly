设计模式：PrintSpooler 单例与扩展为三个实例
原题要求
评阅教师 得分 二、设计模式（共 20 分）。 1. Which design pattern is applied in the code snippet below? 2. Draw UML diagram for this pattern. 3. Revise the code if we need 2 more objects of class PrintSpooler. public class PrintSpooler { private static final PrintSpooler instance = new PrintSpooler(); private PrintSpooler() {} public static PrintSpooler getInstance() { return instance; } }
参考答案20 分
解析与实现

原代码使用 eager singleton：private static final instance + private constructor + public static getInstance()。

若需要总共 3 个 PrintSpooler 对象，可以把唯一实例改为固定对象池，并通过 getInstance(int index) 或轮询方法返回。

public class PrintSpooler {
    private static final PrintSpooler[] INSTANCES = {
        new PrintSpooler(), new PrintSpooler(), new PrintSpooler()
    };
    private PrintSpooler() {}
    public static PrintSpooler getInstance(int index) {
        if (index < 0 || index >= INSTANCES.length) {
            throw new IllegalArgumentException("index must be 0..2");
        }
        return INSTANCES[index];
    }
}
// UML: PrintSpooler -- private static INSTANCES[3], private constructor, public static getInstance(index)
S2
编程题：手机公司月账单管理
原题要求
评阅教师 得分 三、编程题（共 40 分）。 Write a working Java program that manages monthly billing information for a cell phone company's customers. In crafting your solution, you can use the Input class that is supplied (the JavaDoc API is shown below). Overview of the Requirements All customers are charged a base rate for their phone plan, but they will face additional charges based on their choice of plan. There are two types of plans: • Per Minute Plan: In addition to the base rate, customers will pay a per-minute charge of 25 cents. • Flat Rate Plan: In addition to the base rate, customers will pay an extra premium fee which can vary for each customer. Details: Hierarchy of Phone Classes There are 3 class: Phone, PerMinutePhone and FlatRatePhone. The Phone Class The Phone class will be abstract. It will have the following instance variables all of which must be private: • phoneNumber: holds the phone number in a String object. There is no need for any validation of this field during input. • baseRate: holds a floating-point number. This value is entered during keyboard input. • taxAmt: holds the calculated tax. This value is calculated when all customer bills are displayed. The Phone class will also hold a constant for the HST tax (which is now going to be 13%). The PerMinutePhone and FlatRatePhone Classes Both PerMinutePhone and FlatRatePhone are subclasses of Phone. The PerMinutePhone class has the following private instance variable: • minutes: holds a floating-point number which represents the total number of minutes used during that month. The FlatRatePhone class has the following private instance variable: • flatRatePremium: holds the additional cost of having a flat-rate plan. The Class Methods The organization of methods is largely up to you, with the following requirements. You must implement overridden methods (virtual methods) to enter values from the keyboard, and to calculate the before-tax bill total. The tax calculation can be left as an exclusive responsibility of the Phone class. Details: Customer Class The Customer class will hold the customer's name in a String object, and have a reference to a Phone object. It will require suitable methods to perform: • input • processing / calculations • display of output Details: PhoneCompany Class The PhoneCompany class is used to create an object, then the addCustomer() and calcBills() methods are called as a result of a user's menu choice. The addCustomer() method will create a new Customer object and create the correct type of Phone object based on the user's choice. The PhoneCompany class must ensure that the addition of new customers will fit in the collection that is used to manage all the added customers. The calcBills() method will determine the billing total for each customer, then display the results. Methods Available in Input Class. Details: TestPhoneCompany Class Implement the TestPhoneCompany class. The following screen capture demonstrates a sample run of the program. You can see that two customers have been added, then the bills are displayed for those two customers. Of course, more customers could have been added. Class Input Method Summary static int getInt(String prompt) Retrieves an int from the keyboard. static int getInt(String prompt, int low, int high) Retrieves an int from the keyboard which is guaranteed to be between the specified ranges (inclusive). static int getDouble(String prompt, double low, double high) Retrieves a double from the keyboard which is guaranteed to be between the specified ranges (inclusive). static String getString(String prompt) Retrieves a String from the keyboard where the text is delimited by the <Enter> key.
参考答案40 分
解析与实现

核心设计是抽象父类 Phone 保存公共字段和税额计算；PerMinutePhone 与 FlatRatePhone 重写税前金额计算；Customer 组合一个 Phone；PhoneCompany 维护客户集合。

下面代码给出一份可运行参考实现，输入工具类可用 Scanner 代替题面 Input 类。

import java.util.*;

abstract class Phone {
    private String phoneNumber;
    private double baseRate;
    private double taxAmt;
    public static final double HST = 0.13;

    public void input(Scanner in) {
        System.out.print("Phone number: ");
        phoneNumber = in.nextLine();
        System.out.print("Base rate: ");
        baseRate = Double.parseDouble(in.nextLine());
    }

    public abstract double beforeTaxTotal();

    public double total() {
        taxAmt = beforeTaxTotal() * HST;
        return beforeTaxTotal() + taxAmt;
    }

    public String billLine() {
        return phoneNumber + " before tax=" + beforeTaxTotal()
            + ", tax=" + taxAmt + ", total=" + total();
    }

    protected double getBaseRate() { return baseRate; }
}

class PerMinutePhone extends Phone {
    private double minutes;
    public void input(Scanner in) {
        super.input(in);
        System.out.print("Minutes: ");
        minutes = Double.parseDouble(in.nextLine());
    }
    public double beforeTaxTotal() { return getBaseRate() + minutes * 0.25; }
}

class FlatRatePhone extends Phone {
    private double flatRatePremium;
    public void input(Scanner in) {
        super.input(in);
        System.out.print("Flat-rate premium: ");
        flatRatePremium = Double.parseDouble(in.nextLine());
    }
    public double beforeTaxTotal() { return getBaseRate() + flatRatePremium; }
}

class Customer {
    private String name;
    private Phone phone;
    public void input(Scanner in) {
        System.out.print("Customer name: ");
        name = in.nextLine();
        System.out.print("Plan 1=per-minute, 2=flat-rate: ");
        int type = Integer.parseInt(in.nextLine());
        phone = type == 1 ? new PerMinutePhone() : new FlatRatePhone();
        phone.input(in);
    }
    public void displayBill() {
        System.out.println(name + ": " + phone.billLine());
    }
}

class PhoneCompany {
    private ArrayList<Customer> customers = new ArrayList<Customer>();
    public void addCustomer(Scanner in) {
        Customer c = new Customer();
        c.input(in);
        customers.add(c);
    }
    public void calcBills() {
        for (Customer c : customers) c.displayBill();
    }
}

public class TestPhoneCompany {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        PhoneCompany company = new PhoneCompany();
        while (true) {
            System.out.print("1 add, 2 bills, 0 exit: ");
            int choice = Integer.parseInt(in.nextLine());
            if (choice == 1) company.addCustomer(in);
            else if (choice == 2) company.calcBills();
            else break;
        }
    }
}

The Circle method draw should return “Circle-based Drawing”. public class Circle extends Shape { public Circle(String initialName) { // to be implemented } // method draw to be implemented // if the argument obj is a circle and has the same name, return true public boolean equals(Object obj) { } }

import java.util.*;

class Dictionary {
    private ArrayList<String> myDictionary = new ArrayList<String>();
    public boolean hasWord(String word) {
        return myDictionary.contains(word);

AutoConfig：解析 servlet 配置并写入 web.xml
原题要求
评阅教师 得分 三、应用题（本大题共 1 小题，共 20 分）。 You are required to implement a program named AutoConfig to config a servlet information into a file called web.xml. The application prompts the user to "Add a servlet configuration:", the user enters a line with the following format: description_display-name_servlet-name_servlet-class Where: description is about the servlet description. display-name is the displaying name of the servlet. servlet-name is the registering name of the servlet. servlet-class is the class name of the servlet. The fields are delimited by an underscore ( _ ). The application parse user's input and save the configuration information in XML. E.g., a file with the following content should be created: <web-app version="2.4"> <servlet> <description>A Servlet Controller</description> <display-name>Controller</display-name> <servlet-name>Controller</servlet-name> <servlet-class>Controller</servlet-class> </servlet> </web-app> If the user's input is invalid, the application displays an error message.
参考答案20 分
解析与实现

输入按下划线分成 4 段；段数不为 4 或字段为空即为非法。合法时生成 web.xml。

import java.io.*;
import java.util.*;

public class AutoConfig {
    public static void main(String[] args) throws IOException {
        Scanner in = new Scanner(System.in);
        System.out.println("Add a servlet configuration:");
        String line = in.nextLine();
        String[] parts = line.split("_", -1);
        if (parts.length != 4) {
            System.out.println("Invalid input");
            return;
        }
        for (String p : parts) {
            if (p.trim().isEmpty()) {
                System.out.println("Invalid input");
                return;
            }
        }
        PrintWriter out = new PrintWriter(new FileWriter("web.xml"));
        out.println("<web-app version=\"2.4\">");
        out.println("  <servlet>");
        out.println("    <description>" + parts[0] + "</description>");
        out.println("    <display-name>" + parts[1] + "</display-name>");
        out.println("    <servlet-name>" + parts[2] + "</servlet-name>");
        out.println("    <servlet-class>" + parts[3] + "</servlet-class>");
        out.println("  </servlet>");
        out.println("</web-app>");
        out.close();
    }
}

import java.util.*;

class Interest {
    private String name;
    private String description;
    public Interest(String name, String description) {
        this.name = name; this.description = description;
    }
    public String getName() { return name; }
    public void setName(String name) { this.name = name; }
    public String getDescription() { return description; }
    public void setDescription(String description) { this.description = description; }
    public String toString() { return name; }
}

class Faculty {
    private String name, email;
    private ArrayList<Interest> interests;
    private ArrayList<Course> courses;
    public Faculty(String name, String email, ArrayList<Interest> interests, ArrayList<Course> courses) {
        this.name = name; this.email = email; this.interests = interests; this.courses = courses;
    }
    public String getName() { return name; }
    public String getEmail() { return email; }
    public ArrayList<Interest> getInterests() { return interests; }
    public ArrayList<Course> getCourses() { return courses; }
    public String toString() {
        return "Faculty [name=" + name + ", email=" + email
            + ", interests=" + interests + ", courses=" + courses + "]";
    }
}

class Course {
    private String name, syllabus;
    private Faculty mainInstructor;
    private int credit;
    public Course(String name, Faculty mainInstructor, int credit, String syllabus) {
        this.name = name; this.mainInstructor = mainInstructor;
        this.credit = credit; this.syllabus = syllabus;
    }
    public String toString() {
        return "Course [name=" + name + ", " + mainInstructor
            + ", credit=" + credit + ", syllabus=" + syllabus + "]";
    }
}
