Add 5 string objects in the set in order: "Zhang three", "Li four", "Wang five", "horse six", "Zhao Qi". Traverse the ArrayList to print the position and content of each element in the set, and then print the size of the set. 参考答案40 分
解析与实现

四题分别考察字符串遍历、数组计数、异常处理和 ArrayList 基本操作。下面给出可直接拆分为独立类的参考代码。

// 1. Count uppercase letters
Scanner in = new Scanner(System.in);
String s = in.nextLine();
int count = 0;
for (int i = 0; i < s.length(); i++) {
    if (Character.isUpperCase(s.charAt(i))) count++;
}
System.out.println(count);

// 2. Digit frequency with BufferedReader
BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
String line = br.readLine();
int[] freq = new int[10];
for (int i = 0; i < line.length(); i++) {
    char ch = line.charAt(i);
    if (Character.isDigit(ch)) freq[ch - '0']++;
}
for (int i = 0; i < freq.length; i++) {
    System.out.println(i + ": " + freq[i]);
}

// 3. Max/min of 10 integers
Scanner input = new Scanner(System.in);
int max = Integer.MIN_VALUE, min = Integer.MAX_VALUE;
for (int i = 0; i < 10; i++) {
    try {
        int v = input.nextInt();
        max = Math.max(max, v);
        min = Math.min(min, v);
    } catch (InputMismatchException ex) {
        throw new IllegalArgumentException("input is not an integer");
    }
}
System.out.println("Max=" + max);
System.out.println("Min=" + min);

// 4. ArrayList<String> names
ArrayList<String> names = new ArrayList<String>();
Collections.addAll(names, "Zhang three", "Li four", "Wang five", "horse six", "Zhao Qi");
for (int i = 0; i < names.size(); i++) System.out.println(i + ": " + names.get(i));
System.out.println("size=" + names.size());
String third = names.remove(2);
System.out.println("deleted third=" + third);
System.out.println("size=" + names.size());

原题要求
三、UML 设计题（本大题共 1 小题，共 20 分） 评阅教师 得分 Draw a UML Class Diagram representing the following elements from the problem domain for a hockey league. A hockey league is made up of at least four hockey teams. Each hockey team is composed of six to twelve players, and one player captains the team. A team has a name and a record. Players have a number and a position. Hockey teams play games against each other. Each game has a score and a location. Teams are sometimes lead by a coach. A coach has a level of accreditation and a number of years of experience, and can coach multiple teams. Coaches and players are people, and people have names and addresses. Draw a class diagram for this information, and be sure to label all associations with appropriate multiplicities. Notes: captain could alternatively be represented as a second, named association between player and team. Assumptions: each player only plays on one team, each captain only captains one team, each team only plays in one league.

  - position: String

  - accreditationLevel: String
  - yearsExperience: int
HockeyTeam
  - name: String
  - record: String
Game
  - score: String
  - location: String
